# Linux Namespace

* 很多编程语言都包含了命名空间的概念，我们可以认为命名空间是一种封装，封装本身实际上实现了代码的隔离
* 在操作系统中命名空间命名空间提供的是系统资源的隔离，其中系统资源包括了：进程、网络、文件系统......

linux系统实现命名空间主要目的之一就是为了实现轻量级虚拟化服务，达到隔离的目的
